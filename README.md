# Victor Iglesias — Instructional Design Portfolio

Modern dashboard-style portfolio built with Next.js (App Router), TypeScript, Tailwind CSS, and Framer Motion.

## Quick Start

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## Structure

- Home, Projects, Project detail, About routes
- Content data in src/content/projects.ts
- Reusable UI components in src/components
